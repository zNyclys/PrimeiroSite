Básico da Programação HTML

Tags Iniciais

!DOCTYPE html

Devemos começar nosso código escrevendo o tipo do documento para que ele seja identificado corretamente.
- Não precisa ser fechada com "/".


html

Ao colocarmos a tag "html", damos início a nossa linha de código.
- Todo o nosso código deve estar dentro desta tag.
- Devemos fechar a tag usando "/".


head

A tag "head" serve para escrevermos informações do nosso site para o código, informações estas que não serão visíveis no conteúdo do site ao acessá-lo.
- Devemos fechar a tag usando "/".


body

A tag "body" é o "corpo" do site. São as informações que serão vísiveis no site, como textos, cores do site ou detalhes visuais.
- Devemos fechar a tag usando "/".


Tags Diversas


Aba do Navegador


title

A tag "title", quando dentro da "body", define um nome para a guia aberta no navegador.
- Devemos fachar a tag usando "/"
- Fica dentro da "body".


link

A tag "link", quando dentro da "body" e criado com atributos, pode ser utilizado para definir a imagem na guia do navegador.
Ex: link rel="shortcut icon" type="imagex/png" href="D:\icon.ico"
